Matlab code for my blog post on ERP monsters:
<https://garstats.wordpress.com/2016/06/03/erp-monsters/>

`make_monsters.m` can be used to generate data similar to those in the post, using the real ERP data stored in `erps.mat`.

`monsters_behind_barplots.m` generates all the figures presented in the post, using data stored in `monsters.mat`.

## Dependencies
To run the code, you will need functions from these toolboxes:

<http://www.mathworks.com/matlabcentral/fileexchange/54243-univarscatter>

<http://www.mathworks.com/matlabcentral/fileexchange/42919-pink--red--blue-and-violet-noise-generation-with-matlab-implementation/content/pinknoise.m>

<https://github.com/LIMO-EEG-Toolbox/limo_eeg>
